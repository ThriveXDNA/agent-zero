## Integration Patterns

### Lead Generation Pipeline
**Goal:** Capture, enrich, and route leads efficiently

**Workflow:**
1. **Clay/Close/Hunter** → Enrich lead data
2. **ActiveCampaign** → Add to nurture sequence
3. **HubSpot** → Assign to sales rep and track pipeline
4. **GA4** → Track lead acquisition channels
5. **Slack/Buzz** → Notify team of high-quality leads

**Key Data Flow:**
- Email capture → Validation → Enrichment → Scoring → Distribution → Analytics

**Goal:** Distribute content across platforms consistently

## Troubleshooting Guide

### Common Issues and Solutions

**Issue: Integration not receiving data**
- Check API credentials
- Verify webhook URLs are accessible
- Review rate limits
- Check for CORS issues
- Validate data format

**Issue: Data not syncing correctly**
- Review field mappings
- Check data transformation rules
- Verify API version compatibility
- Review error logs
- Test with sample data

**Issue: Webhooks not firing**
- Verify webhook registration
- Check webhook endpoint status
- Review payload format
- Test with sample payload
- Check for firewall issues

**Issue: Analytics not tracking**
- Verify tracking code installation
- Check event names match
- Review GA4 data stream settings
- Test with user interactions
- Check for ad blockers

**Issue: Performance issues**
- Implement caching
- Review API call frequency
- Optimize data queries
- Use batch operations
- Review pagination
