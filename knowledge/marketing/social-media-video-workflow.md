# **MULTI-PLATFORM VIDEO WORKFLOW**

### **How to Create One Video and Distribute Across All Platforms**

## **INTRODUCTION**

This document provides a streamlined workflow for creating video content
once and adapting it across all social media platforms. This approach
maximizes efficiency while ensuring each platform receives optimized
content.

Reference Documents:

\- For detailed video production: See
\`VIDEOPRODUCTIONCHECKLISTWITHWHY.md\`

\- For CapCut editing: See \`ThriveXDNACapCutPro_Guide.docx\`

\- For platform-specific rules: See individual platform SOPs

## **THE CORE PRINCIPLE**

Create Once, Distribute Everywhere

Instead of creating separate content for each platform, follow this
workflow:

1\. Create one master video

2\. Adapt it for each platform\'s requirements

3\. Optimize for each platform\'s algorithm

4\. Publish across all platforms

This saves 60-70% of production time while maximizing reach.

## **PHASE 1: PLANNING (30-60 minutes)**

### **Step 1: Select Multi-Platform Topic**

☐ Choose a topic that works across platforms:

\- \[ \] Has broad appeal (not too niche)

\- \[ \] Can be shortened for short-form platforms

\- \[ \] Has visual potential for all formats

\- \[ \] Fits multiple platform audiences

Topic Examples That Work:

\- Quick tips (work for TikTok, Shorts, Reels)

\- Educational content (work for YouTube, LinkedIn)

\- Behind-the-scenes (work for Instagram, TikTok, Facebook)

\- Myth-busting (universal appeal)

### **Step 2: Plan the Master Video**

☐ Design the core video content:

\- \[ \] Write script that can be shortened/expanded

\- \[ \] Plan hooks that work for multiple formats

\- \[ \] Include B-roll that works across platforms

\- \[ \] Design for both horizontal (16:9) and vertical (9:16)

The 3-Video Rule:

Instead of one long video, plan to create:

1\. Master version (2-5 min): Full content for YouTube Long-form,
LinkedIn

2\. Short version (30-60 sec): Condensed for TikTok, Shorts, Reels

3\. Micro version (15-30 sec): Ultra-short for Stories, Twitter

### **Step 3: Prepare Multi-Platform Assets**

☐ Gather all assets before filming:

\- \[ \] Write versions for different lengths

\- \[ \] Plan multiple hook options

\- \[ \] Prepare all B-roll needed

\- \[ \] Create text overlays for each platform

\- \[ \] Plan platform-specific CTAs

## **PHASE 2: PRODUCTION (60-90 minutes)**

### **Step 4: Record the Master Footage**

☐ Follow VIDEOPRODUCTIONCHECKLISTWITHWHY.md

Record more than you need:

\- \[ \] Record full master version first

\- \[ \] Record extra B-roll angles

\- \[ \] Capture multiple hook options

\- \[ \] Get cutaway shots for transitions

Pro Tip: Record Vertical AND Horizontal

If possible, record key sections in both 16:9 and 9:16:

\- \[ \] Set up horizontal camera for YouTube/LinkedIn

\- \[ \] Set up vertical camera/phone for TikTok/Reels/Shorts

\- \[ \] Capture same content in both formats

### **Step 5: Create Multiple Versions**

☐ In editing, create all versions:

Version 1: YouTube Long-Form (5-15 min)

\- Full script with complete value

\- Horizontal 16:9 format

\- Professional structure with intro/body/CTA

\- Custom thumbnail

Version 2: Short-Form (30-60 sec)

\- Condensed version with strongest hooks

\- Vertical 9:16 format

\- Fast-paced, high energy

\- Captions required

Version 3: Micro version (15-30 sec)

\- Single key point or tip

\- Vertical 9:16 format

\- Ultra-fast hook

\- Perfect for Stories

## **PHASE 3: EDITING & EXPORT (60-90 minutes)**

### **Step 6: Edit the Master Version**

Follow VIDEOPRODUCTIONCHECKLISTWITHWHY.md

For YouTube Long-Form:

\- \[ \] Use full script

\- \[ \] Include all key points

\- \[ \] Add custom thumbnail

\- \[ \] Structure with intro, body, CTA

\- \[ \] Add chapters/timestamps

\- \[ \] Export as MP4 (H.264), 1080p or 4K

### **Step 7: Create Short-Form Versions**

For TikTok, YouTube Shorts, Instagram Reels, Facebook Reels:

☐ Edit vertical version (9:16):

\- \[ \] Select best hooks from multiple takes

\- \[ \] Cut to 30-60 seconds

\- \[ \] Add captions (every word)

\- \[ \] Use trending audio if applicable

\- \[ \] Add text overlays synced to audio

\- \[ \] Ensure fast pacing (cuts every 3-5 seconds)

☐ Export settings for each platform:

TikTok: 1080x1920, MP4, H.264, captions burned in

YouTube Shorts: 1080x1920, MP4, H.264, captions burned in

Instagram Reels: 1080x1920, MP4, H.264, captions burned in

Facebook Reels: 1080x1920, MP4, H.264, captions burned in

### **Step 8: Create Micro Versions**

For Instagram Stories, Twitter/X:

☐ Create 15-30 second version:

\- \[ \] Single key point or tip

\- \[ \] Fastest hook possible

\- \[ \] Vertical 9:16 format

\- \[ \] Add captions

\- \[ \] Export as MP4

## **PHASE 5: PUBLISHING**

### **Step 11: Publishing Schedule**

Best Practice: Stagger Your Posts

Instead of posting everywhere at once, stagger for maximum visibility:

Morning Block (8-10 AM CT):

\- \[ \] YouTube Shorts (9 AM)

\- \[ \] LinkedIn post (if scheduled)

\- \[ \] First round of tweets

Midday Block (12-1 PM CT):

\- \[ \] TikTok video

\- \[ \] More tweets

Evening Block (6-8 PM CT):

\- \[ \] Instagram Reels (6 PM)

\- \[ \] YouTube Shorts (second)

\- \[ \] Final tweets

Note: Each platform\'s algorithm rewards fresh content. Staggering helps
maximize reach without competing with yourself.

### **Step 12: Cross-Posting vs. Native Upload**

When to Cross-Post:

\- Same video works for multiple platforms

\- Content is performing well - reuse on other platforms

\- Quick distribution when time is limited

When to Create Unique Content:

\- Platform-specific trends

\- Different audience expectations

\- Significantly different format works better

The Best Approach:

\- Create 2-3 variations of the same core content

\- Post to each platform natively

\- Repost top performers to other platforms

## **EFFICIENCY TIPS**

### **Speed Up Production**

1\. Batch Record:

\- Record multiple videos in one session

\- Use same background/setup

2\. Template Your Edits:

\- Create caption templates

\- Use consistent text styles

\- Pre-made transitions

3\. Use Smart Export Presets:

\- Save export settings for each platform

\- Batch export all versions at once

4\. Repurpose Smartly:

\- Create vertical first (works everywhere)

\- Crop horizontal from vertical if needed

\- Don\'t over-edit each version

5\. Schedule in Advance:

\- Schedule posts for optimal times

\- Use platform scheduling tools

\- Batch your publishing
