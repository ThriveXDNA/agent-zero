Fonts to use
If you’re not sure which fonts to pick, don’t experiment. Go
with safe choices like:
Poppins, Satoshi, Plus Jakarta Sans, Inter.

Icons to use
I attached a .fig file with a set of 100 of our own icons you
can use for your project, but you can also download many
of the free icon packs from the internet. A few things to
think about though:
- Just use ONE set of icons. Don’t combine multiple packs
as the icons won’t be consistent.
- Make sure the icons are not too complex visually - simple
is good.

Project ideas
My project is a SaaS web application that helps a
restaurant chain manage orders and drivers. You can use
the overall idea and just modify the industry - for example
do it for a plumbing store and service business, with
plumbers driving to fix clients problem. Or managing a
chain of bookstores, music band merchandise, courier
services etc.
If you want to try something completely different you can
try creating a cryptocurrency exchange (so no delivery and
drivers), an email client, a secure messaging service, a
portal for amateur bands to show their music, a travel site,
hotel booking service and more.
All of these ideas require tables, charts, dashboards and
forms, so you can use the same building blocks. Pro tip: try
to create something from an industry you’re personally
interested in - that always yields the best results! :)
